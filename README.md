# Smart-Ecommerce-analytics-platform
A production-style e-commerce analytics platform built as a 4-week data science capstone. The project covers exploratory data analysis, customer segmentation (RFM + K-Means), churn prediction (Random Forest), weekly revenue forecasting (LSTM), and a product recommendation engine (collaborative filtering).All results are presented through an interactive Streamlit dashboard. Dataset: UCI Online Retail II (~1M transactions).
